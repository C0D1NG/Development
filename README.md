# 👉 Clone the repository
# 👉 Create a branch
# 👉 Add your projects in a new directory, the name of directory should be your ```USERNAME```
# 👉 Add & commit
# 👉 Push changes to GitHub
# 👉 Compare & Submit a Pull Request
